template<class T>
T TNT::hypot(const T &a, const T &b)¶

Computes the hypotenuse of real (non-complex) scalars a and b by avoiding underflow/overflow using:

$a * sqrt( 1 + (b/a) * (b/a)))$

rather than

$sqrt(a*a + b*b)$

Return

hypotenuse of type T

Template Parameters

  • T: Input data type, must be the “real” part of a number

Parameters

  • a: first number

  • b: second number
