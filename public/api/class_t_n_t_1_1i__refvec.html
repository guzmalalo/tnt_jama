template<class T>
class TNT::i_refvec¶

Internal representation of ref-counted array. The TNT arrays all use this building block.

If an array block is created by TNT, then every time an assignment is made, the left-hand-side reference is decreased by one, and the right-hand-side refernce count is increased by one. If the array block was external to TNT, the refernce count is a NULL pointer regardless of how many references are made, since the memory is not freed by TNT.
