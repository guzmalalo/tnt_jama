template<class Real>
Real TNT::hypot(const Real &a, const Real &b)¶

Return

hypotenuse of real (non-complex) scalars a and b by avoiding underflow/overflow using (a * sqrt( 1 + (b/a) * (b/a))), rather than sqrt(a*a + b*b).
