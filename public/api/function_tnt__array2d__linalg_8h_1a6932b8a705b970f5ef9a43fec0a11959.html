Function Documentation¶

template<class T>
Array1D<T> TNT::sum(const Array2D<T> &A, int i)¶

Returns an Array1D containing the sum in a given direction of the Array2D.

Example: 

double a_values[3][3] = {{2, 7, 6},
                        {9, 5, 1},
                        {4, 3, 8}};
TNT::Array2D<double> A(3, 3, *a_values);

TNT::Array1D<double> sum_col = TNT::sum(A,2); // 15 x 3
TNT::Array1D<double> sum_row = TNT::sum(A,1); // 15 x 3
Return

T Sum: the sum of the elements in the direction 1

Template Parameters

  • T: Type od data

Parameters

  • A: Input array

  • i: direction 1: rows, 2: columns.
