template<class T>
bool TNT::near(const Array2D<T> &A, const Array2D<T> &B, T tol)¶

Verify if the all the values of A are near to the values of, value per value given a tolerance.

Return

true if abs(A-B) > tol

Return

false if at least one value is different

Template Parameters

  • T: data type

Parameters

  • A: First array to compare

  • B: Second Array to compare

  • tol: tolerance for the comparaison
