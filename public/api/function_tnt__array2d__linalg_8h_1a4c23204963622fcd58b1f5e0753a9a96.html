template<class T>
Array2D<T> TNT::matmult(const Array2D<T> &A, const Array2D<T> &B)¶

Matrix Multiply: compute C = A*B, where C[i][j] is the dot-product of row i of A and column j of B.

Return

the (m x k) array A*B, or a null array (0x0) if the matrices are non-conformant (i.e. the number of columns of A are different than the number of rows of B.) Example: 

TNT::Array2D<double>
C = TNT::matmult(A, B);

Parameters

  • A: an (m x n) array

  • B: an (n x k) array
